幻灯片jquery.SuperSlide插件

引入jQuery 和 jquery.SuperSlide.js

  1. <script type="text/javascript" src="http://lib.h-ui.net/jquery/1.9.1/jquery.min.js"></script> 
  2. <script type="text/javascript" src="http://lib.h-ui.net/jquery.SuperSlide/2.1.1/jquery.SuperSlide.min.js"></script>

  1. <div id="slider-3">
  2. 	<div class="slider">
  3. 		<div class="bd">
  4. 			<ul>
  5. 				<li><a href="#" target="_blank"><img src="images/banner-1.jpg" ></a></li>
  6. 				<li><a href="#" target="_blank"><img src="images/banner-1.jpg" ></a></li>
  7. 				<li><a href="#" target="_blank"><img src="images/banner-1.jpg" ></a></li>
  8. 				<li><a href="#" target="_blank"><img src="images/banner-1.jpg" ></a></li>
  9. 			</ul>
  10. 		</div>
  11. 		<ol class="hd cl dots">
  12. 			<li>1</li>
  13. 			<li>2</li>

CSS代码

  1. .slider{position:relative;text-align:center; margin:0 auto;z-index:1;}
  2. .slider .bd,.slider .bd li,.slider .bd img{width:960px; height:272px}/*请给每个幻灯片套个div并设置id，通过id重置这个地方的宽度，达到自定义效果*/
  3. .slider .bd{z-index:2;overflow:hidden}
  4. .slider .bd li{float:left;width: 100%;overflow:hidden; background-position:center; background-repeat:no-repeat}
  5. .slider .bd li a{ display:block; width: 100%; height: 100%}
  6. .slider .bd li img{display:block}
  7. .slider .hd{ position: absolute; z-index: 3; left: 0; right: 0; bottom:10px; padding: 0 10px; text-align: center}
  8. .slider .hd li{display:inline-block;text-align:center;margin-right:10px;cursor:pointer;background-color:#C2C2C2}
  9. .slider .hd li.active{background-color:#222}
  10.   /*圆点*/
  11.   .dots li{width:10px; height:10px;font-size:0px;line-height:0px;border-radius:50%}
  12.   /*数字*/
  13.   .numbox li{width:20px; height:20px; line-height:20px; font-size:13px;font-family:Arial;font-weight:bold; text-indent:inherit;}
  14.   .numbox li.active{color:#fff}
  15.   /*长方条*/
  16.   .rectangle li{width:40px; height:10px;font-size:0px;line-height:0px}

默认hd上面追加的dots为圆点，将dots改为numbox就变成带数字的方框，rectangle为不带数字长方条

  1. #slider-3{ width: 960px; height: 273px; overflow: hidden}/*这个是demo页面定义尺寸的样式，请根据自己幻灯片的尺寸重新定义。切莫生搬硬套。*/

js 代码

调用代码：

  1. $(function(){
  2. 	jQuery("#slider-3 .slider").slide({mainCell:".bd ul",titCell:".hd li",trigger:"click",effect:"leftLoop",autoPlay:true,delayTime:700,interTime:7000,pnLoop:false,titOnClassName:"active"});
  3. });

宽屏幻灯片效果，要想撑满屏幕，建议图片宽度做成2000px以上，中间主要视觉区间控制在网页的宽度之内，两边过渡到纯色，图片通过li上背景的方式去实现。这样不管是宽屏的还是窄屏的都可以很好的展示。具体效果见： lib/jquery.SuperSlide/2.1.1/demo/02/demo.html

相关参数

参数 版本 默认值 说明 详细
type 2.1 slide 效果类型，可选"slide"||"menu"，引入type:"menu"专门处理菜单/导航效果

mouseOverStop 2.1 true 鼠标移到容器层（无缝滚动是mainCell）是否停止播放

switchLoad 2.0 > 2.1 null 内容切换加载，暂时只支持图片，必须配合后台程序使用。当为图片的时候switchLoad为图片实际路径的属性名称，例如switchLoad:"_src"
[v2.1] 修复当scroll>1时，switchLoad无效的bug。 		详解

[v2.1]增加传递参数，startFun:function( i, c, slider, titCell, mainCell, targetCell, prevCell, nextCell ){}，这样简写就方便了。
例如 jQuery(".slideBox").slide({ startFun:function( i,c,s ){ s.show() } }); 这里的 s.show() 相当于 jQuery(".slideBox").show() 		详解

 详解
triggerTime 2.0 150 毫秒，titCell触发时间，默认150毫秒延迟，以防重复执行效果。若为0即鼠标移过titCell立刻执行效果
