本服务仅支持ie8+以及其它支持html5的浏览器

用户前端使用说明：

$("#testBtn").fnImageCrop({
		aspectRatio : 4 / 3,
	accno: accno,//必传
	key:  random,//必传
	sign: md5(random+"6799f4e52a01461fa463b3ed81e79e0c"),//必传
	flash_swf_url : '/imagecut/resources/js/Moxie.swf',//必传
	silverlight_xap_url : '/imagecut/resources/js/Moxie.xap',//必传
	},function(res){//返回处理后图片，并处理的回调函数
		if(res.status == "success"){
			$("#resultImg").attr("src", res.preview).show();//返回预览地址有效期一天，过期自动删除
		}else{
			alert("出错了");
			window.location.reload();
		}
	}
);

options 参数说明（可选）


参数名 默认值 参数说明
allowSelect true 允许新选框
allowMove true 允许选框移动
allowResize true 允许选框缩放
aspectRatio 0 选框宽高比。说明：width/height
dragEdges true 允许拖动边框
minSelect [0,0] 选框最小选择尺寸。说明：若选框小于该尺寸，则自动取消选择
maxSize [0,0] 选框最大尺寸
minSize [0,0] 选框最小尺寸
accno '' 用户帐号（必填项）
key '' 随机数（必填项）
sign '' 认证签名（必填项，由随机数和用户key生成md5签名）
ImgType "gif,jpeg,jpg,bmp,png" 上传图片类型(逗号之间不能有空格)
max_file_size "2mb" 上传图片大小
flash_swf_url '' 图片上传使用flash（必填项)
silverlight_xap_url '' 图片上传使用xap文件（必填项)
